The Oscilloscope infinniVision DSO-X 3014T is available at Wirid-Lab as a remote instrument.

Follow next steps to access the device:

  1. Reserve node 75 with the application: (ARM) VNC + Jupyter

2. Test the response of the oscilloscope using the ping command in a terminal window.
ping  192.168.0.2     

3. Run the VNC :

          Write xvnc4viewer in a terminal window. 
          server address (Write the Oscilloscope`s address): 192.168.0.2
